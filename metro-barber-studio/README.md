# Metro Barber Studio

Welcome to the Metro Barber Studio project! This website is designed to showcase our barber shop's services, gallery, and booking options with a bold, urban aesthetic.

## Project Structure

The project is organized as follows:

```
metro-barber-studio
├── public
│   ├── index.html          # Main HTML document
│   └── favicon.ico         # Favicon for the website
├── src
│   ├── assets
│   │   └── fonts           # Custom fonts for the urban aesthetic
│   ├── components
│   │   ├── Header.jsx      # Navigation bar and logo
│   │   ├── Footer.jsx      # Copyright and social media links
│   │   ├── Services.jsx    # Services offered by the barber shop
│   │   ├── Gallery.jsx     # Showcase of images
│   │   └── BookingForm.jsx # Online appointment booking form
│   ├── pages
│   │   ├── Home.jsx        # Landing page
│   │   ├── About.jsx       # Information about the barber shop
│   │   ├── Services.jsx    # Detailed services page
│   │   ├── Gallery.jsx      # Gallery page
│   │   └── Contact.jsx      # Contact information and form
│   ├── styles
│   │   └── main.css        # Main CSS styles
│   └── App.jsx             # Main application component
├── package.json             # npm configuration file
└── README.md                # Project documentation
```

## Features

- **Bold Urban Design**: The website features a modern and edgy design that reflects the atmosphere of Metro Barber Studio.
- **Service Listings**: Easily view all the services we offer.
- **Gallery**: Browse through our gallery to see our work and the shop's vibe.
- **Online Booking**: Book your appointments directly through the website.
- **Responsive Design**: The website is designed to be fully responsive for all devices.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd metro-barber-studio
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
